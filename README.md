# Setup

## Mandatory steps
1. Install Python3.9+
2. Install Pipenv

## Setup project
Install environment
```bash
# Create virtual environment
pip install pipenv

#pipenv install --dev
pipenv shell
```

## Run Project in Docker
```bash
docker-compose up -d 
```

## Formatting code
```bash
# Black
pip install black
python -m black
```
